第六題

同上題，請問 console.log( ) 會顯示什麼？

  1. 112.5
  2. 250
  3. 50
  4. "total" is read-only
點我看答案 

答案: 4  


在嚴謹模式若使用 freeze
Object.freeze(wallet);  


而且還另外賦值
wallet.save = 500;  


因為 Object.freeze 會去凍結這個物件，所以就無法去改變物件原有的total
因此出現 read only 的錯誤
