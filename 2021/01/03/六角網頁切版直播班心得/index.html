六角網頁切版直播班心得

نُشر في 2021-01-03 عُدل في 2021-01-04

前言

紀錄六角網頁切版秋季班共八週的課堂心得。

當初為什麼報名這堂課？

在今年年初開始了我的轉職人生，雖然有上過實體的前端課程，但是結業後還是覺得自己學的不夠扎實，所以在網路找線上課程。因緣際會下找到了六角學院，買了(HTML/CSS/jQuery/RWD…)，認真的看了課程也有配合實作，但是不知道自己的實力與業界是不是差異甚大，或是自己的學習的規劃是不是合適的，於是報名了網頁切版直播班，
透過老師有系統化的教學與鞭策，還有同學間的交流討論，讓自己成長。

學習過程

第一週 1 px 也不差的版型控制術

雖說對於 HTML 及 CSS 有基礎的概念也會運用，但是真的要做到 1 px 也不差也是一門學問呀！人家都說易學難精，在這週對我來說算是再複習 HTML 標籤元素屬性，以及加深容器的觀念。

第二週 - Flexbox 網頁排版術

這週開始有些難度囉！在課前老師都會先釋出預習的影片，如果有跟著老師的節奏走，在課堂中就能夠比較快跟上腳步。本週的 Flexbox 透過遊戲的方式，可以讓自己更加了解對於 Flexbox 的熟練度。

第三週 - 響應式網頁設計

在上直播班以前，也有學過響應式網頁要怎麼寫，但是你知道權重很重要嗎？不要亂用 id / important ，除了觀念導正之外，課堂中也分享了許多業界在撰寫程式的眉角與技巧，還有要怎麼規劃斷點，這些都是讓我少走冤忘路的寶貴意見。

第四週 - 多頁式網頁設計

這週的直播學習開始導入 Gulp 自動化流程和 SCSS 系統，課堂中聽老師講解操作好像很簡單，但是要自己從頭到尾實際走一遍才知道原來沒這麼簡單好嗎！
好家在老師與助教有提供手冊給學員，還有課程加碼的助教分享，讓我們能容易上手！

第五週-後臺表單設計

這週開始導入了 Bootstrap 結合 SCSS ，看了同學的 code 開始寫 OOCSS BEM 之類的，我滿臉問號？(噎 ~ 我們確定是上同一門課嗎？)，然後心理莫名的著急起來，不過後來老師也有說如果還不熟，就先不要碰 OOCSS BEM ，等自己熟悉了再去了解會比較好，所以心理也調適了不少。

第六週- Bootstrap 格線系統與元件整合

這週讓我更了解格線系統是什麼，它的原理是什麼，比較重要的是元件這部分，是我從未學習到的，透過同學在 slack 發問，還有觀看別人寫的 code ，能夠讓自己對於元件的拆分更了解。

第七週-視差滾動

來到這週算是可以稍微輕鬆一些，學一些動畫效果，讓網頁看起來更酷炫一點。老師也透過獎品來鼓勵願意認真學習的學員。

第八週- CSS 模組化

時間過得很快，不知不覺就來到最後一週了，我的作業要加快腳步了！在這週主要是統整前七週的觀念，學習 SCSS 結構再進化、容器與內容分離，自己也慢慢的朝向這個方向努力。

最大的收獲是？

  • 作業有人幫忙 code review ，讓我一次一次更精進程式碼的品質

  • 資源很多，可以看到很厲害的同學的 code ，有問題隨時在 slack 上發問，
    老師/助教/同學都會很熱情的回應。

  • 透過每週的作業、小組任務以及訓練菜單，這樣大量的密集訓練，讓你能夠在一拿到版型時，就能夠馬上進入狀況，知道怎麼去設計跟規劃每個區塊。

身為學長姐，分享些想入坑的新同學一些勉勵的話

如果你也想學好切版，六角的切版直播版真的可以投資，認真的花 2 ~ 3 個月的時間，專注在老師給的訓練菜單及作業上，你會發現真的進步很多，而且這邊的資源甚至是你在業界也可能學不到的，在六角的大家都是你學習上的導師，六角的學員是來自各個不同職業背景，甚至是已經在職但是想要更精進自己的來學習，透過他們身上可以學習到很多。

在學習過程中你可能會心急，覺得跟不上？怎麼大家都這麼強？不過沒關係，每個人都會有自己的進度與學習目標，所以你只要針對自己的部分做努力，然後盡可能地跟上老師、跟上大家就可以了。踏實地把每一步走穩進步才會快，反而急著學習而草草了事，這樣就失去了意義。
